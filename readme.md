# ORB-SLAM2-ROS

This repo is a simple demo of ORB-SLAM2 in ROS. Data generated from SLAM, such as camera pose, keyframe trajectory and pointcloud are published in ROS.  

* DBOW2 and G2O should be compiled independently.
